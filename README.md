## 🚦 Accident Severity Prediction

This project predicts **road accident severity** using a **Linear Regression model** built with **Python and Scikit-Learn**.  
It demonstrates the complete machine learning workflow including **data preprocessing, feature selection, model training, and evaluation**.

## ⚙️ Tech Stack
- Python
- Pandas
- NumPy
- Scikit-Learn
- Jupyter Notebook

## 🔍 Project Workflow
1. Data loading and preprocessing  
2. Feature selection and cleaning  
3. Train-test split  
4. Model building using Linear Regression  
5. Model evaluation using performance metrics  

## 📊 Dataset
US Accidents dataset from Kaggle used for training and testing the model.

## 🚀 Goal
To build a simple machine learning model that can help analyze and predict the **severity level of road accidents** based on available features.
